解题 leetcode

  • 2014-09-22

眼见进入九月份，就要开始找工作了，赶紧开始复习 C++，顺便也要找题目练习，于是找到了 leetcode。 leetcode 题目难度不大，但是其中涉及到了很多基础知识，而且 leetcode 本身就是定位面试。 还有最重要的一点，就是题量少，只有 151 题，赶紧刷完，以后终于也能说我也刷过一整套题库了 :)

我都是用 C++ 写的，现在已经完成了 115/151，代码放在 github@adousj/leetcode 上。

刚开始激情澎湃，事情也少，进度还算可以，这几天各种事儿又纷至沓来，就慢了下来，争取十一结束前能搞定吧。

C++ 编译器使用的 GCC 4.7.2，大部分 C++11 的功能是支持的，我想可以使用些 auto lambda 智能指针，来看看 C++11 到底是一门多新的语言。
