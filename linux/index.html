linux 创建用户命令

  • 2014-07-18
  • linux， command

因为从来都是自己用自己的电脑，而且一直使用的是Ubuntu，所以都是在系统安装过程中新建用户，从来没有尝试过用命令行。

这次使用过程中涉及到的命令如下：

useradd
adduser
usermod
userdel
passwd

useradd

useradd adou

只会创建 adou 用户，但是并没有创建 /home/adou/ 文件夹，也没有创建密码，不能登录系统。

useradd -m adou

这种方式是可以同时创建 /home/adou/，不过密码还是需要通过:

passwd adou

设置

adduser 添加用户

adduser adou

这个命令更适合我这种初学者，一个命令全部搞定。

　usermod 添加至用户组

用户建立了之后要需要权限的，特别是 sudo 权限，默认有 sudo 用户组，把新用户加入即可。

usermod -a -G sudo adou

userdel 删除用户

userdel adou

这个只会删除用户，并不会删除用户文件。 如果想连同文件全都删除的话

userdel -r adou
