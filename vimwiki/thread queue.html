python thread queue

threading 模块为线程提供了一个高级接口。源自 java 的线程实现，与低级的 thread 模块相同，只有在编译解释器时打开线程支持才可以使用它。 只需要继承 thread 类，定义好 run 方法，就可以创建一个新的线程。 使用时创建该类的一个或多个实例，然后调用 start 方法。每个实例的 run 方法运行在他自己的线程中。
