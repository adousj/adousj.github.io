rpm 安装

ubuntu的软件包格式是deb，如果要安装rpm的包，则要先用alien把rpm转换成deb。68SLinux联盟 sudo apt-get install alien #alien默认没有安装，所以首先要安装它68SLinux联盟 sudo alien xxxx.rpm #将rpm转换位deb，完成后会生成一个同名的xxxx.deb68SLinux联盟 sudo dpkg -i xxxx.deb #安装68x联盟 注意，用alien转换的deb包并不能保证100%顺利安装，所以可以找到deb最好直接用deb

bz2安装

解压：tar jxvf ***.bz2 打补丁： 1.现在假设你把你补丁的压缩文件解压到/patch目录下，解压后的文件名为：xxxx.patch 2.然后进入你内核源文件的文件夹,假设为:/root/linux-2.6.x/ cd /root/linux-2.6.x/ 3.进行打补丁，命令如下：patch -Np1 -i /patch/xxxx.patch 4.成功。 PS：解压后最好看看有没有帮助文档吧，例如README。linux的软件一般都有的。
