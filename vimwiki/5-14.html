删除后挽救的

把视频的前后两部分，分别配上不同的音乐，为什么一定要把视频分开配上音乐，然后再合成一个视频呢？ 可以把两个音频合成一个，然后和视频配成整体，计算量小啊。 中间需要用到cat命令，windows没有，用emacs的eshell完成，cat视频停住，或者是太慢，时间太长。 CMD :

ffmpeg -i music1.mp3 -sameq -ar 44100 aout1.mp3 ffmpeg -i music2.mp3 -sameq -ar 44100 aout2.mp3

eshell中：

cat aout1.mp3 aout2.mp3 >aout.mp3

CMD :

ffmpeg -i video.avi -i aout3.mp3 result.avi

ffmpeg

一、ffmpeg命令详解 ffmpeg非常强大，轻松几条命令就可以完成你的工作。

把darkdoor.[001-100].jpg序列帧和001.mp3音频文件利用mpeg4编码方式合成视频文件darkdoor.avi： $ ffmpeg -i 001.mp3 -i darkdoor.%3d.jpg -s 1024×768 -author skypp -vcodec mpeg4 darkdoor.avi

ffmpeg还支持mov格式： $ ffmpeg -i darkdoor.%3d.jpg darkdoor.mov

要查看你的ffmpeg支持哪些格式，可以用如下命令： $ ffmpeg -formats | less

还可以把视频文件导出成jpg序列帧： $ ffmpeg -i bc-cinematic-en.avi example.%d.jpg

debian下安装ffmpeg很简单： ＃apt-get install ffmpeg
